# Pour ou Contre App With Expo

[![made with expo](https://img.shields.io/badge/MADE%20WITH%20EXPO-000.svg?style=for-the-badge&logo=expo&labelColor=4630eb&logoWidth=20)](https://github.com/expo/expo) [![supports iOS and Android](https://img.shields.io/badge/Platforms-Native-4630EB.svg?style=for-the-badge&logo=EXPO&labelColor=000&logoColor=fff)](https://github.com/expo/expo)

L'application est codée en React Native avec la librairie Expo qui permet d'exporter le projet sous Android ou Ios et faire des tests d'interface via l'application Android ou Iphone

## Smartphone side

Télécharger l'application Expo et créer un compte

## Install & Build

Installation :

Node.js LTS release

Expo :  `npm install -g expo-cli`

Yarn

Lancement du projet :
Lancer dans le dossier du projet:

`yarn` or `yarn install`

`expo install`

Puis `expo start` 

Test interface :

Sur le serveur local généré par Expo, passer en mode tunnel puis scanner le QRcode avec l'application Expo
